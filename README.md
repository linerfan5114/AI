## 🎯 Overview

This is a machine learning project that predicts **computer prices** based on hardware specifications.  
Simply enter five key specs — **CPU**, **RAM**, **SSD**, **GPU**, and **Brand** — and the model will estimate the market price in USD.

Perfect for:
- 🛒 **Buyers** — Know if a computer is fairly priced
- 🏪 **Sellers** — Get quick, accurate pricing for listings
- 📊 **Students** — Study a complete ML pipeline from data to prediction

---

## 🧠 Model

| Detail | Value |
|---|---|
| **Algorithm** | XGBoost Regressor |
| **Tuning** | GridSearchCV (5-fold cross-validation) |
| **Number of Features** | 5 input + one-hot encoding |
| **Training Data** | `computer_specs_very_large.csv` |

### Input Features

| Feature | Description | Example |
|---|---|---|
| **CPU** | Processor brand & model | `Intel i7`, `AMD Ryzen 5` |
| **RAM_GB** | RAM size in GB | `16` |
| **SSD_GB** | SSD storage in GB | `512` |
| **GPU** | Graphics card | `NVIDIA RTX 3070`, `GTX 1650` |
| **Brand** | Computer manufacturer | `Dell`, `Lenovo`, `Apple` |

### Output

| Output | Description |
|---|---|
| **Price_USD** | Predicted price in US Dollars |

## 🚀 Installation & Usage

### 1. Clone the repository
```bash
git clone https://github.com/linerfan5114/Computer-Price-Predictor.git
cd Computer-Price-Predictor
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train the model
```bash
python train.py
```
This will:
- Load & preprocess the dataset
- Train XGBoost with GridSearchCV
- Save the best model as `model.pkl`

### 4. Make predictions
```bash
python predict.py
```
Enter specs interactively and get price estimates instantly.

## 🔧 How It Works

### Training Pipeline (`train.py`)
1. **Load** `computer_specs_very_large.csv`
2. **Clean** numeric columns (RAM_GB, SSD_GB, Price_USD)
3. **Normalize** categorical columns (CPU, GPU, Brand → lowercase)
4. **One-hot encode** categorical features
5. **Split** data (80% train / 20% test)
6. **Tune** XGBoost with `GridSearchCV` (5-fold CV)
7. **Save** best model + metadata to `.pkl` files

### Prediction Pipeline (`predict.py`)
1. **Load** trained model and metadata
2. **Accept** user input interactively
3. **Standardize** GPU/CPU/Brand names via mapping tables
4. **Validate** input against known training values
5. **Encode** input to match training feature columns
6. **Predict** price and display result
